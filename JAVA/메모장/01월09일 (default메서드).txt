인터페이스의 멤버
1) 상수
2) 추상 메서드
3) default 메서드 (인터페이스에 default 메서드를 만들어)
4) 클래스(static) 메서드

<<default메서드>>
등장 이유 : 인터페이스 변경이 있을시 인터페이스의 추상메서드를 선언하면
	    구현한 클래스들에 다시 추가한 메서드를 구현해야함.

public default 리턴타입 메서드명(매개변수) {}


<<static메서드>>
등장 이유 : default 메서드가 구현부가 있는것이 인터페이스에 추가되었으니,
	    static 메서드도 추가되었음
	   (제외된 이유는, 자바를 좀더 쉽게 배우기 위한 규칙이었을뿐이었다.)
