객체지향언어(OOP)=>Oriented Object Program
특징 :  1) 기존의 프로그래밍과 크게 다르지 않다.
	2) 코드의 재사용성이 높다
	3) 코드의 관리가 쉬워졌다.
	4) 신뢰성이 높은 프로그램의 개발을 가능하게 된다.

//객체는 클래스에서 정의된 내용대로 메모리에 할당된다.//

객체 VS 인스턴스
=> 거의 비슷한 의미나 객체가 조금더 넓은 의미이다.



<클래스를 구성하는 3가지 요소>

1) 멤버변수 =>클래스 내부에 선언된 변수
2) 멤버메서드 =>클래스 내부에 선언된 메서드
3) 생성자 => 매개변수가 없는 생성자는 기본생성자라고 불린다.

클래스 내부의 멤버변수는 자동으로 초기화된다.
String => null로 초기화
boolean => false로 초기화
int => 0으로 초기화


static 변수는 class영역에 들어가미

변수 => 	1) static변수(클래스 변수)
	2) instance 변수
	3) 지역변수

static을 함부로 바꾸면 안된다.(공유변수)
